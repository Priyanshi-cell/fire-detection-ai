# 🔥 Fire & Smoke Detection using YOLOv8

## 📌 Overview

This project is an AI-powered system that detects **fire and smoke in real-time** using computer vision and deep learning.

It uses a trained YOLOv8 model with OpenCV to process video input and trigger alerts using sound when fire or smoke is detected.

---

## 🚀 Features

* 🔍 Real-time fire & smoke detection
* 🎥 Works on video input
* 🔊 Audio alert system using Pygame
* ⚡ Fast detection using YOLOv8
* 🧠 Custom trained model (`.pt`)

---

## 🛠️ Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Pygame

## ▶️ How to Run

### 1. Clone the repo

```
git clone https://github.com/Priyanshi-cell/fire-detection-ai.git
cd fire-detection-ai
```

### 2. Install dependencies

```
pip install ultralytics opencv-python pygame
```

### 3. Run the script

```
python fire_hf.py
```
